What protocols belong to the Application layer?

The Application Layer provides network services to user applications.

Common Protocols:

HTTP – Web browsing
HTTPS – Secure web browsing
FTP – File transfer
SMTP – Sending emails
POP3 / IMAP – Receiving emails
DNS – Domain name to IP address conversion
Telnet – Remote login
SSH – Secure remote login



One-Line Interview Answer

The Application layer includes protocols such as HTTP, HTTPS, FTP, SMTP, POP3, IMAP, DNS, Telnet, and SSH.
