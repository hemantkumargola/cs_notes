Explain the Layers of the TCP/IP Model

The TCP/IP Model has 4 layers, and each layer has a specific role in data communication.

1. Application Layer
Provides network services to user applications.
Combines the functions of the OSI Application, Presentation, and Session layers.
Protocols: HTTP, HTTPS, FTP, SMTP, DNS, Telnet.

Example: Opening a website in a browser using HTTP/HTTPS.

2. Transport Layer
Provides end-to-end communication between devices.
Ensures reliable or fast data transfer.
Protocols:
TCP – Reliable, connection-oriented.
UDP – Faster, connectionless.

Example: TCP is used for file downloads, while UDP is used for video streaming and online gaming.

3. Internet Layer
Responsible for logical addressing and routing of data packets.
Finds the best path from source to destination.
Protocols: IP, ICMP, ARP.

Example: IP helps route data from one computer to another over the internet.

4. Network Access Layer
Handles the physical transmission of data over the network.
Deals with MAC addresses, cables, Wi-Fi, and Ethernet.
Technologies: Ethernet, Wi-Fi, PPP.

Example: Sending data through a Wi-Fi router or Ethernet cable.
