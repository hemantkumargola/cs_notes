### What is CIDR? (Easy Interview Answer)

**Answer:**

**CIDR (Classless Inter-Domain Routing)** is a method of assigning IP addresses more efficiently. It uses **prefix notation** (such as `/24`) to indicate how many bits belong to the network portion of an IP address.

**Example:** `192.168.1.0/24`

### Short Interview Answer

> **CIDR is a method of IP address allocation that uses prefix notation (e.g., `/24`) to identify the network portion of an IP address and reduce IP address wastage.**
